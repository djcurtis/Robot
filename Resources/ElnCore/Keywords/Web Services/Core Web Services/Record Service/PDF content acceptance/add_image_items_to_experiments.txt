Resource          add_items_to_experiments.txt

*** Keywords ***
Add JPEG Item To Experiment
    [Arguments]    ${experiment_id}    ${data_file_location}    ${item_type}=Uploaded JPEG document
    [Documentation]    Adds a jpeg to the experiment and version save
    ...
    ...
    ...    *Arguments*
    ...
    ...    ${experiment_id} id of exp or record to put it in
    ...
    ...    ${data_file_location} \ where the data file is picked up
    ...
    ...    ${item_type} - optional, set to item type property to be displayed (can be anything you like, it doesn't have to be to dow ith file type etc)
    ...
    ...    *Precondition*
    ...    Experiment must exist
    ...
    ...    *Return value*
    ...    Entity id of the document created
    ...
    ...
    ...    *Example*
    ${doc_entity_id}=    Add DOCUMENT To Experiment    ${experiment_id}    ${data_file_location}    image/jpeg    JPG_IMAGE    ${item_type}
    [Return]    ${doc_entity_id}
